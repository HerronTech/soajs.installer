Welcome to SOAJS Installer

Hello Beaver and welcome to the SOAJS installer.
This Wizard will guide you through the steps needed to customize and install a fresh copy of SOAJS on your local or remote machine.

Prerequisites:

Before you proceed, make sure that the following technologies are installed depending on the type of deployment you want:


Manual Installation
Ubuntu: 
# Install NodeJs, NPM, MongoDB & Nginx
$ sudo apt-get update && sudo apt-get install -y npm nodejs mongodb nginx
$ sudo ln -s /usr/bin/nodejs /usr/bin/node
# Start MongoDB
$ sudo service mongodb start

Mac: 
# Install Brew
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew update
$ brew upgrade
$ brew services

# Install NodeJS, MongoDB, Nginx
$ brew install homebrew/versions/node012 mongodb nginx
# Start MongoDB
$ brew services start mongodb
